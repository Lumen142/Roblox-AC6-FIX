
# AC6-FIX v1.0.0

“A-Chassis Tune” automatically kicks players from the server if they have audio playback issues in certain versions.

# Roadmap

### Note: If you use ROJO, it will be added automatically.

## If you are not using ROJO:

- Open Roblox Studio.
- Open your project.
- Create a “Script” named “AC6-FIX” under “ServerScriptService”.

## Then:

- Go to the code in src/server/init.server.luau.
- Copy the code and paste it into the “Script” you created.
## FAQ

#### Which versions does it work with?

The system has currently only been tested on version “1.4.” However, if your vehicles have the ‘AC6C_Boost_Sound’ and “AC6C_Stock_Sound” LocalScript files, this system will likely work on your vehicles.
